### БЛОК 1: HERO (First Screen)

**Цель:** Захватить внимание за 3 секунды. Передать суть предложения.

**Контент:**

```
[Бейдж] 60 участников | 12+ ниш | Второй поток

[H1] Один человек. Ноутбук. $20/мес на подписки.
Результат = отдел из 12 специалистов.

[H2] Практический воркшоп «AI-Маркетолог»: за 6 недель собери
работающую AI-систему маркетинга — от исследования до первого клиента.

[CTA] Выбрать пакет →
[Подпись] от $297 · рассрочка до 9 платежей · 100% возврат
```

**Дизайн:**
- Полноэкранный блок, тёмный фон
- Заголовок — крупный, bold, с градиентом на ключевых словах
- Под CTA — строка доверия (иконки: Mono, PayPal, USDT, Visa)
- Справа или снизу — анимированная иллюстрация или скриншот (лендинг, сделанный через Claude Code)

### БЛОК 2: БОЛЬ «ЗНАКОМО?»

**Цель:** Вызвать узнавание. Читатель должен кивнуть: «Это про меня».

**Контент:**

```
[H2] Знакомо?

☐ Платишь дизайнеру $300 за лендинг и ждёшь 2 недели
☐ Тратишь 3 часа на контент-план — а результат «как у робота»
☐ Клиент написал: «Мы решили попробовать сами. С ChatGPT»
☐ Подписалась на 30 каналов про AI — каша в голове стала ещё гуще
☐ Хочешь масштабироваться, но физически не можешь вести больше 3 клиентов
☐ Купила курс за $300 → посмотрела 2 урока → забросила → ненавидишь себя

[Текст] Если хотя бы 2 пункта — про тебя, читай дальше.
Потому что через 6 недель всё это можно заменить одной системой.
```

**Дизайн:**
- Чекбоксы (интерактивные — можно кликать, визуальный отклик)
- Каждый пункт — на отдельной строке
- Акцентные выделения на болезненных словах
- Плавный скролл к следующему блоку

### БЛОК 3: КОНТРАСТ «БЫЛО → СТАЛО»

**Цель:** Визуально показать трансформацию. Продать «пункт назначения».

**Контент:**

| Раньше (без AI) | Теперь (с AI-системой) |
|---|---|
| $5 000–$10 000 на запуск | $20–$200/мес на подписки |
| 2–3 недели на лендинг | 2 часа голосом |
| 3 часа на контент-план | 30 минут на неделю |
| 5–12 подрядчиков | 1 человек |
| Зависимость от команды | Полная автономность |
| Потолок: 3 клиента | 10 проектов в одиночку |

**Дизайн:**
- Два столбца: красный/серый (боль) vs зелёный/акцентный (результат)
- На мобильном: вертикальный формат, сначала «раньше», потом «теперь» с анимацией перехода
- Зачёркнутые цифры слева, яркие — справа

### БЛОК 5: ПРОГРАММА (6 МОДУЛЕЙ)

```

[Модуль 1] 🔬 Deep Research
Находишь нишу, аудиторию, конкурентов за 1 день.
3 промта × 5 нейронок = полная картина рынка.
→ На выходе: стратегия + GAP-анализ + «кнопка бабло»

[Модуль 2] 📦 Упаковка и оффер
ДНК Клиента + оффер по Хормози.
AI-ассистент, который знает твою ЦА лучше тебя.
→ На выходе: упакованный оффер + позиционирование

[Модуль 3] 🎨 Визуальный контент
AI-генерация: Nano Banana Pro, Kling, видеокреативы.
Контент-план на месяц за 30 минут.
→ На выходе: готовый контент на 4 недели

[Модуль 4] 💻 Вайб-кодинг
Лендинг голосом через Claude Code. Квизы, калькуляторы, боты.
Деплой на Vercel — бесплатно.
→ На выходе: работающий лендинг + лид-магнит

[Модуль 5] 🤖 AI-агенты
Ассистент продаж, автоответы, аналитика.
Кастомные скиллы для Claude Code.
→ На выходе: AI-ассистент для твоего бизнеса

[Модуль 6] 🚀 Воронки и запуски
Вебинарная, трипваерная, классическая.
AI-ориентированные микро-продукты.
```

**Дизайн:**
- Вертикальный таймлайн / аккордеон
- Каждый модуль — раскрывающаяся карточка
- Иконка + название + описание + «На выходе» (выделено)
- Прогресс-бар внизу: Модуль 1 → ... → Модуль 6 → 🎯 Результат

## 3. Дизайн-система

### Цвета
| Роль | Цвет | Код |
|---|---|---|
| Фон | Почти чёрный | #0a0a0a |
| Текст основной | Белый | #f5f5f5 |
| Текст вторичный | Серый | #a1a1aa |
| Акцент 1 (CTA) | Ярко-зелёный | #22c55e |
| Акцент 2 (выделения) | Фиолетовый | #8b5cf6 |
| Градиент | Фиолетовый → синий | #8b5cf6 → #3b82f6 |
| Боль (контраст) | Красный-приглушённый | #ef4444 с opacity 0.7 |
| Успех (контраст) | Зелёный | #22c55e |

### Типографика
| Роль | Шрифт | Размер (mobile/desktop) |
|---|---|---|
| H1 | Inter Bold | 28px / 48px |
| H2 | Inter SemiBold | 24px / 36px |
| Body | Inter Regular | 16px / 18px |
| Small | Inter Regular | 14px / 14px |
| CTA | Inter Bold | 18px / 20px |

### Компоненты
- **CTA-кнопка:** скруглённая (8px), зелёный фон, белый текст, hover-эффект (яркость +10%)
- **Карточка:** тёмный фон (#1a1a2e), border 1px #2a2a3e, скругление 12px, padding 24px
- **Бейдж:** маленький pill, фиолетовый фон, белый текст
- **Аккордеон FAQ:** border-bottom, иконка + / −, плавное раскрытие
- **Sticky CTA (mobile):** фиксированная кнопка внизу экрана, backdrop-blur фон

## 4. Технические требования

### Performance
- First Contentful Paint < 1.5s
- Largest Contentful Paint < 2.5s
- Cumulative Layout Shift < 0.1
- Шрифты: preload Inter (woff2)
- Изображения: WebP, lazy loading
- Видео: YouTube embed (не автоплей)

### SEO
- Title: «AI-Маркетолог: Завод в кармане за 6 недель | Влад Яско»
- Description: «Практический воркшоп по AI-маркетингу. Один человек заменяет отдел из 12 с помощью AI. 60 участников первого потока. От $297.»
- OG-image: 1200×630, тёмный фон + заголовок + фото Влада

### Аналитика
- Google Analytics 4
- Meta Pixel
- UTM-параметры на CTA-кнопках
- Scroll depth tracking (блоки 25/50/75/100%)
- Click tracking на CTA

### Интеграции
- Форма оплаты: ссылка на внешнюю платёжку (Mono / Stripe / WayForPay)
- Telegram-бот: кнопка «Написать в Telegram» → ссылка на бота или личку Влада
- Лид-магнит: popup или inline-форма для email/Telegram

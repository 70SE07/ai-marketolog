## Создание ассистента "ДНК Клиента"

~15 мин

Создаём ИИ-ассистента, который будет анализировать вашу аудиторию. Весь контекст и инструкции уже готовы — нужно только скопировать и вставить.

Не нужно разбираться в маркетинге — ассистент знает всё за вас.

[🧠 Claude](https://claude.ai/) [✨ Gemini](https://gemini.google.com/)

### 1.1 Вариант A: Claude Project (рекомендуемый)

📋 Пошаговая инструкция

1 Откройте claude.ai → Projects → Create Project

2 Название: «ДНК Клиента (by Vlad Yasko)»

3 Нажмите "Get Instructions" ниже → скопируйте ИНСТРУКЦИЮ → вставьте в поле Instructions проекта

5 Готово! Откройте чат внутри проекта и начните работать

📚 Что внутри: структура ассистента

Инструкция (Instructions)

Системный промт, который задаёт роль, правила и формат работы ассистента. Копируется в поле Instructions проекта.

Части 1-5 (файлы)

Методология анализа ЦА, разбитая на 5 частей: основы, сегментация, портреты, боли/триггеры, язык клиента. Загружаются как файлы.

Руководство

Детальное руководство по работе с ассистентом и интерпретации результатов.

## Путь A: CustDev — реальные данные

~3-5 дней (сбор) + 30 мин (анализ)

Если у вас есть аудитория (подписчики, клиенты, база) — CustDev даёт самые точные данные. Реальные цитаты клиентов = готовые заголовки.

Реальные цитаты клиентов = готовые заголовки для рекламы.

[🔧 Google Forms](https://docs.google.com/forms) [🧠 Claude](https://claude.ai/)

Скопировать

Ассистент с загруженным ДНК Бизнеса персонализирует анкету автоматически.

💡

Совет

Отправляйте анкету через Telegram-канал, email, посты в соцсетях. Минимум 10 ответов для анализа.

🎙 2.2 Глубинные интервью

🎙 Запись

Записывайте на YouTube (unlisted) или Zoom. Автоматические субтитры = готовый транскрипт.

⏱ Время

30-45 минут на интервью. Больше — устанет, меньше — не раскроется.

💬 Слушайте

80% времени говорит собеседник, 20% — вы. Не продавайте.

🔍 Копайте

«Расскажите подробнее», «А что вы имеете в виду?», «Почему?»

⚡

Лайфхак

5-10 интервью достаточно. Загружайте каждый транскрипт как файл в ассистента «ДНК Клиента».

## Финальная формула

~5 мин

Зафиксируйте результат. Одна аудитория + одна боль + один продукт = готовая гипотеза.

1 Аудитория + 1 Боль + 1 Продукт

= Готовая гипотеза для запуска

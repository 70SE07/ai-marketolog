## ПАТТЕРНЫ ЦА ИЗ ЧАТА

### Эмоциональный цикл ученика
1. **Вау-эффект** (первые 2 недели) — «ахуеваю от возможностей»
2. **Перегрузка** (недели 2-3) — «мозг начал плавиться», «инфы очень много»
3. **Техно-барьеры** (недели 3-4) — «скиллы не устанавливаются», «ошибки»
4. **Прокрастинация** (недели 4-5) — «откладываю задачи», «не могу заставить себя»
5. **Прорыв** (неделя 6+) — «наконец установила всё», «меня прорвало»

### Что реально ценят
1. **Транскрибация голосом** — самый часто обсуждаемый инструмент
2. **Claude Code** — «прям разработчиком себя чувствую»
3. **NotebookLM** — используют как «мозг» для всех документов
4. **Вайб-кодинг лендингов** — конкретный, измеримый результат
5. **Комьюнити** — поддержка группы критически важна

### Что вызывает трудности
1. **Установка скиллов** — запутались где что и как ставить
2. **Лимиты подписок** — ищут бесплатные/дешёвые варианты
3. **Слишком много инструментов** — не понимают что выбрать
4. **Прокрастинация** — особенно у СДВГ-типов
5. **Техно-пробелы** — «я гуманитарий», ошибки пугают

## КЛЮЧЕВЫЕ СМЫСЛЫ ДЛЯ УПАКОВКИ 2-ГО ПОТОКА

### Смыслы по продукту

1. **«80% запросов решает»** (цитата Татьяны) — мощное числовое обещание
2. **«Разработчиком себя чувствую»** (Александра) — трансформация идентичности
3. **«Наконец чётко увидела чего хочу»** (Виктория) — не про AI, а про ясность
4. **«Высвободится время на просто жизнь»** (Марина) — продаём свободу
5. **«Энергия Влада — это топ»** (Repko) — энергия ведущего = USP

### Смыслы по аудитории

1. **Главный страх = «не разобраться»** → нужен пошаговый путь
2. **Рассрочка критична** → $297/мес × 2 = порог входа
3. **Комьюнити = якорь** → без группы бросят на 3-й неделе
4. **Практика > теория** → «дайте инструмент и покажите»
5. **СДВГ-паттерн массовый** → геймификация, таймеры, маленькие шаги

### Возражения из чата (для скриптов продаж)

1. «Инфы очень много, немного покороче» → формат: короткие модули
2. «Скиллы не устанавливаются» → нужен онбординг-чеклист
3. «У меня Android/iPad/Windows» → мультиплатформенность
4. «Лимиты ChatGPT/Claude» → обучение оптимизации расхода
5. «Когда будут ответы на вопросы?» → Q&A сессии критичны

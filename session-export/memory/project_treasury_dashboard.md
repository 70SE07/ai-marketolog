## Суть продукта

Система управленческого казначейства для онлайн-бизнесов с несколькими совладельцами, ФОП и счетами.

**Основная боль:** При обороте 1.2М UAH/мес деньги приходят через WayForPay, проходят через цепочку счетов (WFP → ФОП Полозовского → ФОП Лукьяненко → личный счёт → крипто → выплата совладельцу), и без системы невозможно отследить движение и понять сколько кому причитается.

**Цепочка движения денег:**
1. Деньги пришли на WFP (поступления)
2. Поступили на счёт ФОП Полозовского
3. Переведены на ФОП Лукьяненко
4. С ФОП на личный счёт
5. Сняты наличные / куплено крипто
6. Отправлены совладельцу (Владу, Антону)

**До системы:** Игорь Иванович тратил целый день на ручные сверки, допускал ошибки, не мог собрать картину. После — 2% недельной загрузки.

**Как создавалась:** Claude Code + GitHub + MCP-коннектор к Google Sheets. Совет от Карпышина.

## Целевая аудитория

Собственники онлайн-школ с оборотом 200K-2M UAH/мес, 2-3 совладельца, WayForPay, несколько ФОП (Украина).

## Текущий MVP

Google Sheets + Apps Script: 29 листов, 33 скрипт-файла, 259 функций. Автоимпорт WFP, автоначисления, автоудержания, QC-движок, P&L отчёт с PDF-экспортом.

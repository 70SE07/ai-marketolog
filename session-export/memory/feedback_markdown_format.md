Все документы по проекту лендинга/конвертера делать в **Markdown**, не в HTML.

Название продукта: **AI-Маркетолог** (флагманский продукт Влада Яско).
**AI-Ковчег** — это бренд/комьюнити, НЕ название продукта.

**Why:** Влад явно попросил: «дай мне это в Markdown формате. И все предыдущие документы, которые будем делать, не нужно мне их в HTML переводить.»

**How to apply:** При создании любых документов для этого проекта (ДНК, промты, исследования, офферы) — сохранять как `.md`. В тексте документов использовать «AI-Маркетолог» как название продукта.

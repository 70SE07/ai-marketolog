# Tone of Voice: Влад Яско

## Архетип
«Свой пацан, который взломал систему» — не гуру на пьедестале, а парень из окопов.
Нарратив: «Был обычным работягой-маркетологом, решил проблему через нейронки, теперь пропускная способность = целый отдел»

## Характеристики стиля
1. **Разговорный "пацанский"** — как в кафе с другом. Сленг, мат точечно для усиления
2. **Прямолинейность без фильтра** — не прячет провалы и сырость продукта. Уязвимость = доверие
3. **Энергичность и скорость** — всё пронизано движением. Короткие импульсные сообщения
4. **Антиперфекционизм** — «криво, но быстро», «начни хоть как-то»

## Фирменные маркеры
- **Движ** — вселенная = движение
- **Залететь** — не «записаться», импульс
- **Машина** — подтверждение, «погнали»
- **На коленке** — антиперфекционизм, быстрый MVP
- **Гиперценность** — компенсация косяков избыточной пользой
- **Стакать/сметчить** — комбинировать (геймерская культура)
- **Завод в кармане** — core-метафора

## Механика продаж
- Голосовые как основной инструмент (до 13 мин)
- Продажа за ~30 минут от первого сообщения до согласия
- Обработка возражений: техника «Да, и...» (не спорит, расширяет предложение)
- После согласия — ноль болтовни: «Машина» → «Пришли почту»
- Дробные импульсные сообщения: одно слово = один эмоциональный удар

## Эмоциональные триггеры
- Уязвимость («Был в жопе»)
- Свой парень (сленг, нет дистанции)
- Скорость («быстро», «сейчас», «залететь»)
- Антиперфекционизм (разрешение делать криво)
- Эксклюзив (тестовая группа, ограниченные места)
- Безопасность (100% возврат, рассрочки)
- Трансформация («12→1», «месяц→1 день»)

## Правила для контента от имени Влада
1. Простой разговорный язык
2. Короткие импульсные сообщения
3. Честность — слабости → сила
4. Энергия — каждое сообщение толкает вперёд
5. «Начни хоть как-то» вместо «подготовься идеально»
6. Компромисс вместо давления
7. Эмодзи точечно (❤️🔥👍)
8. Мат только для усиления, не фон

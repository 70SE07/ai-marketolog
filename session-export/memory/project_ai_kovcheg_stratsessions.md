# Стратсессии AI-Ковчег

## Формат стратсессий
- Персональный разбор за $800 (пакет со стратегией)
- ~1 час на человека + рефлексия группы
- Влад вытаскивает данные → брейнштормит → принимает решение
- Все участники активно участвуют (обязательно с вебкой)
- NotebookLM используется как база данных по каждому участнику

## Стратсессия 1: Виктория Кова + Марина Дрыгалова
**Виктория Кова** — HR/рекрутинг для эмигрантов в Европе
- Запрос: упаковать оффер, определить ЦА
- Решение Влада: упростить подачу, «понятно, просто, возможно»
- Рефлексия группы: нужна простота, один экран = понятно за 30 секунд

**Марина Дрыгалова** — телесные практики / шибари для пар
- NotebookLM с материалами Марины уже подготовлен
- Показана функция интерактивного подкаста в NotebookLM

type: feedback
---

Пользователь не хочет получать документы в формате Markdown на локальный диск. Вместо этого:
- Сохранять все документы как красиво оформленные HTML-файлы (тёмная тема, Inter, карточки, градиенты)
- Показывать через preview (static-preview сервер на порту 8766)
- HTML должен быть self-contained (встроенные стили, без внешних зависимостей кроме Google Fonts)
- Формат: артефакт, который можно открыть в браузере и показать клиенту

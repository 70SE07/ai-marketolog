# AI Ковчег — Образовательная программа по AI-маркетингу

**Автор:** Влад Яско — ИИ-методолог, предприниматель
**Бренд:** АИ КОВЧЕГ
**Формат:** Онлайн-курс (стримы в Zoom + запись на YouTube), ~1.5 месяца
**ЦА курса:** Предприниматели, эксперты, маркетологи, новички

## Пакеты курса
- Базовый ($20/мес подписка Claude)
- Средний (основной поток)
- Со стратсессией Влада (~$800) — персональная проработка стратегии продукта

## Структура программы (10 уроков)

### 1. ONBOARDING-стрим
- Знакомство, сегментация участников (маркетологи / эксперты с продуктами / "ищущие")
- Цель: каждый должен заработать деньги во время обучения
- Философия: не обучить маркетингу, а дать инструменты которые делают маркетинг за тебя
- Представление команды (Даня Фролов — воронки, и др.)
- "Масса допшин" — техническая настройка инструментов

### 2. Deep Research и стратегический анализ (Урок 1)
- Стек: Perplexity, ChatGPT Deep Research, Gemini, Claude, Parallel AI
- 3 промта × 3+ нейронки = 9-15 запусков параллельно
- Промт 1: Исследование рынка
- Промт 2: Целевая аудитория
- Промт 3: Анализ конкурентов
- Мета-промтинг: ИИ пишет промт для ИИ
- NotebookLM как "Точка сборки" — объединение всех результатов
- GAP-анализ и гипотезы продукта
- Формула "Кнопка Бабло": 1 Аудитория + 1 Боль + 1 Продукт

### Пошаговый гайд Deep Research (отдельный материал на Vercel)
- Шаг 1: ДНК Бизнеса (контекст о себе, 15-30 мин)
- Шаг 2: Мета-промтинг (10 мин)
- Шаг 3: Мультивселенная — запуск DR в параллель (30-40 мин)
- Шаг 4: NotebookLM — точка сборки (15-20 мин)
- Шаг 5: GAP-анализ и гипотезы (20-30 мин)
- Шаг 6: Решение — формула "Кнопка Бабло"
- + 7 маркетинговых блоков (каналы, юнит-экономика, контент-стратегия)
- Сайт гайда: deep-research-guide.vercel.app

### 3. Упаковка продукта (Урок 2 — Маркетинг и упаковка)
- Создание ИИ-ассистента "ДНК Клиента" (Claude Project / Google Gem)
- 6 файлов методологии: Основы анализа ЦА, Сегментация, Портреты, Боли/триггеры, Язык клиента, Руководство
- Путь A: CustDev — реальные данные (анкеты + интервью)
- Путь B: Deep Research — искусственные данные
- Создание ассистента "Offer by Alex Hormozi" ($100M Offers)
- Формула ценности: (Результат мечты × Вероятность) / (Время × Усилия)
- Агрессивный оффер → Адаптация под стиль
- Текстовый прототип лендинга + кросс-чек данными
- Сайт: deep-research-guide.vercel.app/marketing

### 4. Notebook LM урок
- Базовые юзкейсы NotebookLM
- Загрузка материалов (YouTube видео, чаты Telegram, документы)
- Практика на примере участницы Татьяны Хавро (психотерапевт, распаковка)

### 5. Создание лендингов (Урок 3 — Конвертер)
- Вайб-кодинг с Claude Code (десктоп-приложение)
- Claude Opus 4.6 / Sonnet 4.6 как модели
- Создание лендинга из промта за минуты
- Деплой на Vercel
- Добавление интерактивных элементов: калькуляторы, квизы, лид-магниты
- AI-ботики встроенные в сайт
- Философия: "мамочка в декрете сегодня может научиться делать лендинг"
- Программист Дэн — помощник по технической части

### 6. Визуальный контент
- Кейс Jedi Monkey — "Дофаминомика" (симулятор дофамина)
- От праздного любопытства → Deep Research → база знаний → Claude Code → продукт
- 2288 пользователей с одного поста без рекламы
- Nano Banana Pro для визуального контента
- Принцип: любопытство → исследование → прототип → тест → монетизация

### 7-9. Q&A + разбор workflow (3 сессии)
- Разбор домашек участников
- Валидация Deep Research результатов
- GEO (Generative Engine Optimization) — новый тренд
- Создание ассистента для продаж в переписках (на примере Софии Дробининой)
- 10 часов контента → 80 скриншотов → методология → ассистент → приложение (CRM)
- Обратный инженеринг переписок → формула продаж
- Skills в Claude Code для продаж

- "Неважно, насколько вы крутой предприниматель. Правильный алгоритм + инструменты → переиграть экспертов за один вечер"
- "Раньше 2-3 недели и $5,000+ на агентство. Теперь 4-5 часов с ИИ"
- "Не обучить маркетингу, а дать инструменты которые делают маркетинг за тебя"
- "80% результата — правильное решение (стратегия), остальное — исполнение"
- "Делай быстрые понятные легкие действия и потом докручивай"
- Результат курса должен быть выражен в деньгах
